# Drug Manufacturers parser

A list of companies that are manufacturing the commercially available
forms of this drug that are available in Canada and the Unites States.

## Usage

``` r
drug_manufacturers()
```

## Value

a tibble with the following variables:

- generic:

  A list of companies that are manufacturing the generic form of the
  drug.

- url:

  A link to the companies that are manufacturing the drug.

- *drugbank_id*:

  drugbank id
