# Drug Classification parser

A description of the hierarchical chemical classification of the drug;
imported from ClassyFire.

## Usage

``` r
drug_classification()
```

## Value

a tibble with 9 variables:

- description:

- direct-parent:

- kingdom:

- superclass:

- class:

- subclass:

- alternative-parent:

  One or more alternative parents

- substituent:

  One or more substituents

- *drugbank_id*:

  drugbank id
